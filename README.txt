Gerenciador de Anúncios

A aplicação foi desenvolvida utilizando Spring Boot e persistindo em um banco postgres.
Para utilizar o sistema será preciso criar uma base de dados no postgres com o nome "sistema_anuncio".
Em seguida startar o servidor java através do arquivo "AnuncioApplication.java" que está no pacote com.capgemini.ayrton.anuncio
Com o servidor no ar abrir o navegador no endereço http://localhost:8080 onde será chamado o index.hmlt contendo
um uma tabela com os anúncios cadastrados e um link para o cadastro de um novo anúncio.
